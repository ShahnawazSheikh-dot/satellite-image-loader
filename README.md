# Satellite Forest Habitat Analyzer

## Project Description
Processes satellite imagery of forest areas for wildlife habitat analysis.

## Project Structure
```
project/
├── src/forest_processor.py
├── images/raw/
├── images/processed/
├── results/
├── requirements.txt
└── README.md
```

## How to Run
```bash
pip install -r requirements.txt
python src/forest_processor.py
```
